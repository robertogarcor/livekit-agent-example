# Manual de Markdown

## Encabezados
`# H1`, `## H2`, `### H3`, `#### H4`

## Formato de texto
- **Negrita**: `**texto**` o `__texto__`
- *Cursiva*: `*texto*` o `_texto_`
- ~~Tachado~~: `~~texto~~`
- `Codigo inline`: `` `codigo` ``

## Listas
**Desordenada:**
- Item uno
- Item dos
  - Subitem

**Ordenada:**
1. Primer paso
2. Segundo paso

## Enlaces e imagenes
- `[texto](url)` — enlace
- `![alt](url)` — imagen

## Tablas
```
| Columna A | Columna B |
|-----------|-----------|
| Valor 1   | Valor 2   |
```

## Bloques de codigo
Con triple backtick y lenguaje opcional:
```
def hola():
    print("Hola mundo")
```

## Citas
> Texto citado con `> `

## Linea horizontal
Tres guiones: `---`
